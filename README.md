## Features

- Polished real-time style chat interface with:
  - Message bubbles
  - Timestamps
  - Typing indicator
  - Realistic simulated response delays
- Pre-built support conversation flows:
  - Product questions
  - Order status
  - Refund requests
  - Technical support
- Conversation history sidebar for switching between threads
- Widget mode preview:
  - Floating support button
  - Popup chat panel embedded over a website mock
- Settings panel for instant customization:
  - Bot name
  - Avatar initial/emoji
  - Welcome message
  - Brand colors
- Analytics dashboard tab with mock KPI cards and charts:
  - Conversations today
  - Average response time
  - Satisfaction score
  - Resolution rate
- Dark/light mode toggle
- Fully responsive layout for desktop and mobile
- Footer attribution link: Powered by Nackin

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style component architecture
- Recharts
- Lucide React icons
- next-themes

## Getting Started
